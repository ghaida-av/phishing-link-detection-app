# Phishing Link Detection Android App

 Android application for detecting phishing URLs using Machine Learning.



### Prerequisites

- Android Studio 
- Android SDK (API 24+)
- Backend server running 

### Installation

1. **Open Project in Android Studio**
   

2. **Sync Gradle**
   

3. **Configure Backend URL**

  

4. **Run the App**
   

## Usage

1. Enter a URL 
2. Click "Check URL" button
3. View the analysis results:
   Phishing or safe
